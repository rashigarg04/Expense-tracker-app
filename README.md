# 🍃 Marketing Mojito Expense Tracker

A premium and responsive Expense Tracker built using React and Vite.  
This project helps users manage daily expenses, categorize spending, and track totals with a modern glassmorphism-inspired UI.

---

## ✨ Features

- ✅ Add and manage expenses
- ✅ Categorize spending
- ✅ Dynamic spending progress bars
- ✅ Real-time total expense calculation
- ✅ Live currency conversion
- ✅ Responsive design for Desktop & Mobile
- ✅ Dark-themed premium UI with glassmorphism effects
- ✅ Offline fallback mode for API failures

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS3
- Flexbox & CSS Grid
- Frankfurter API

---

## 🌐 API Integration

This project uses the **Frankfurter.app API** for real-time currency exchange rates.

### API Features:
- Live currency conversion
- Real-time exchange rate updates
- Fallback offline mode when API is unavailable

API Used:  
https://www.frankfurter.app/

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/marketing-mojito-expense-tracker.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd marketing-mojito-expense-tracker
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```
